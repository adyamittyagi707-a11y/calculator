# 🧮 Calculator Web App

A simple and responsive Calculator built using HTML, CSS, and JavaScript. This project performs basic arithmetic operations with a clean and user-friendly interface.

## 🚀 Features
- Addition, Subtraction, Multiplication, Division  
- Interactive button-based input  
- Clear (C) and delete (⌫) functionality  
- Error handling for invalid expressions  
- Responsive design  

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript  

## 📂 Project Structure
Calculator/
│── index.html
│── style.css
│── script.js

## ▶️ How to Run
1. Download or clone this repository  
2. Open `index.html` in your browser  

## 📌 Description
This project allows users to perform real-time calculations through an interactive interface. It demonstrates core JavaScript concepts like DOM manipulation, event handling, and dynamic updates.

## 🎯 Purpose
Created as part of a frontend development internship task to practice building interactive web applications.
